Начало работы

Для начала работы с клиентом нужно распаковать все архивы и доустановить игру.

  1. Открываем Rlauncher.exe
  2. Нас просят указать папку с клиентом Reborn. Выбираем распакованную папку \Warface Editing
  3. Далее так же указываем папку к оригинальному Warface (Например D:\GamesMailRu\Warface)
  4. Если всё сделано правильно - начнется распаковка. После завершения напишет "Ok".
  5. Warface Editing\game\GameScriptsC_x64.pak и GameScriptsC_x86.pak нужно удалить!

В дальнейшем Rlauncher понадобится только для обновлений.

Первый запуск

После этих пунктов можно зайти в саму игру \Warface Editing\start.bat

Warface сразу запустит карту Ангар. Интерфейса здесь нет, можно только полюбоваться одной менюшкой на клавишу f7 (Tweak menu).

С первых секунд можно заметить, что в игре видно курсор мыши. Против такого "бага" есть AntiCursor.exe. После открытия программа нас попросит развернуть игру с помощью сочитания клавишь Alt + Tab. Курсор пропадет до закрытия игры.

Программы с названиями 3pers нужны для изменения вида камеры.

В Tweak menu есть все необходимые настройки: fov, gravity, time scale, графика и т.д. Желательно сразу ознакомиться со всем функционалом, поиграться, побегать, пострелять. Для удобства лучше сделать макрос на постоянные нажатия клавиш + и -.

Как добавить бота на карту?

- Заходим в папку уровеня, который нам нужен. Тот же testmap подойдет "Warface Editing\game\levels\testmap". Нам понадобится файл "mission_mission0.xml" из архива "level.pak", открываем и работаем с ним.

Спускаемся вниз, отделяем от остального кода последние 2 строчки, чтобы выглядело примерно так:

</Entity>

- сюда будем вставлять ботов

</Objects>

</Mission>

Вставляем это:

< Name="bot1" Archetype="Debug.PlayerDummy" Pos="0,0,0" Rotate="0,0,0" EntityClass="PlayerDummy" EntityId="5171592" EntityGuid="49D1E7DF919079B6" Layer="Main">

< eiColliderMode="0" EquipPresetName="testBot"/>

</Entity>

Pos="" - ищем через поиск (Ctrl+f) - < Name="SpawnPoint" - и копируем значение Pos в нашего бота

EquipPresetName="testBot" - пресет бота лежит в GameScriptsC.pak\Libs\Config\presets

EntityId="5171592" - нужно писать всегда разное значение, если ботов несколько.
